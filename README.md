### React, Redux, Redux Thunk, React router, Axios, Cypress, ESLint

### Build Setup

``` bash
# install dependencies
npm install

# start server
npm start

# build
npm run build

# run tests
npm run test

# run Cypress E2E tests
npm run cypress:open

```
